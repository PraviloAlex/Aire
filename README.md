# Aire

Минималистичный дыхательный коуч — карманный «переключатель состояния».
Выбираешь состояние (успокоиться, собраться, восстановиться, заснуть…) и дышишь
по ведущему ритму.

**🔗 Живая версия:** https://praviloalex.github.io/Aire/
На телефоне: открой ссылку → меню браузера → «Добавить на главный экран».

## Возможности

- Короткие дыхательные сессии под состояние: спокойствие, фокус, восстановление, сон и др.
- Живой визуальный ритм (орб), таймер фаз, пауза/продолжение, звук и вибрация.
- **Свой ритм** — настраиваешь вдох, задержки и выдох под себя и сохраняешь пресеты.
- Прогресс, короткие уроки, локальные настройки.
- PWA: ставится на главный экран и работает офлайн.

## Польза и осторожность

Aire — wellness-приложение. Оно не ставит диагнозы, не измеряет давление и не
заменяет медицинскую помощь. При сердечно-сосудистых, дыхательных, неврологических
состояниях, беременности и т. п. выбирай мягкие практики и при необходимости
консультируйся со специалистом.

## Технологии

Expo · React Native · TypeScript · expo-router. Web-first PWA.

## Запуск локально

```bash
npm install
npm start          # дев-сервер
# или статическая сборка:
npm run build      # → dist/
npm run preview    # локальный просмотр сборки
```

## Публикация

Сайт выкладывается на GitHub Pages автоматически: при пуше в `main` GitHub Actions
собирает `dist/` и публикует на `https://<логин>.github.io/Aire/`.

## Проверки

```bash
npm run typecheck && npm run lint && npm test
```
